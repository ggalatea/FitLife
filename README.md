Привет! 
Это консольная версия фитнес-бота. 
Фитнес‑бот — это простой инструмент,
короый проведет расчёт базовых показателей: ИМТ и норму воды в день

Для локальной работы бота необходимы некоторые требования. 
Изначально клонировать репозиторий к себе на компьютер через ключ (например, SSH: git clone git@github.com:ggalatea/FitLife.git)
После клонирование необходимо развернуть вернуальное окружение в директории Fitfile и активировать его. 
Затем из файла requirement.txt установить зависимости через команду #pip install -r requirements.txt
Запустить бота можно через команду #python fit_life.py
Удачного пользования!
